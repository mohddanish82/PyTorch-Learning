# PyTorch Autograd Example

## Description

This project demonstrates how PyTorch Autograd automatically calculates gradients.

The program computes the value of a function and its derivative with respect to x.

## Function

y = x^3 + 2x^2 + x

## Concepts Used

- PyTorch
- Tensor
- Autograd
- Gradient Calculation
- Backpropagation

## Run

```bash
python autograd_example.py
```

## Expected Output

```text
4.0
gradient 8.0
```
