# Gradient Descent Using PyTorch

## Description

This project shows how Gradient Descent works using PyTorch.

PyTorch Autograd automatically calculates gradients, and Gradient Descent updates the value of x to minimize the function.

## Function

y = x^4 - 4x^3 + x^2 + 10

## Concepts

- Tensor
- Autograd
- Gradient
- Gradient Descent

## Run

```bash
python gradient_descent.py
```
