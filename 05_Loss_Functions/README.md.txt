# PyTorch Loss Functions

## 📌 What is this?
This file contains simple examples of different loss functions in PyTorch.

Loss function is used to measure the error between predicted values and actual values.

---

## 📂 Files

### mse_loss.py
Example of Mean Squared Error (MSE) loss.

### l1_loss.py
Example of Mean Absolute Error (L1) loss.

### cross_entropy_loss.py
Example of Cross Entropy loss for classification.

---

## 📌 Loss Functions

### 1. Mean Squared Error (MSE)
Used for regression problems.

It calculates the average of squared differences between actual and predicted values.

---

### 2. L1 Loss
Used for regression problems.

It calculates the average of absolute differences between actual and predicted values.

---

### 3. Cross Entropy Loss
Used for classification problems.

It is used when we have multiple classes like cat, dog, etc.

---

## 🚀 How to run

Run each file separately:

```bash
python mse_loss.py
python l1_loss.py
python cross_entropy_loss.py
