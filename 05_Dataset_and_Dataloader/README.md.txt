# PyTorch Dataset and DataLoader

This example demonstrates how to create a custom Dataset and use a DataLoader in PyTorch.

## Concepts

- Dataset
- DataLoader
- Batching
- Shuffling

## Features

- Creates a custom Dataset class
- Implements `__len__()` and `__getitem__()`
- Loads data in batches using DataLoader
- Demonstrates data shuffling

## Requirements

- Python
- PyTorch

## Run

```bash
python dataset_dataloader.py
